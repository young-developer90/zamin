شروع کار

این راهنما شما را در مراحل نصب زمین از منبع، تأیید تنظیمات، نوشتن اولین اسکریپت و سازماندهی پروژه‌های بزرگتر راهنمایی می‌کند.

پیش‌نیازها

قبل از ساخت زمین، مطمئن شوید که موارد زیر روی سیستم شما نصب شده است:

  • Rust 1.80+ — زمین به زبان Rust نوشته شده است. آن را از طریق rustup نصب کنید:
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Git — برای کلون کردن مخزن الزامی است.
  • C compiler — GCC یا Clang در Linux/macOS، MSVC در Windows. برای افزونه‌های بومی و قابلیت‌های اختیاری مورد نیاز است.

نصب

کلون و ساخت

مخزن را کلون کرده و زمین را در حالت release بسازید:

git clone https://github.com/young-developer90/zamin.git
cd zamin
cargo build --release

فایل باینری کامپایل‌شده در target/release/zamin قرار خواهد گرفت. آن را به PATH خود اضافه کنید تا دسترسی راحت‌تری داشته باشید:

export PATH="$PWD/target/release:$PATH"

قابلیت‌های اختیاری

زمین از پرچم‌های قابلیت اختیاری پشتیبانی می‌کند که قابلیت‌های اضافی را فعال می‌کنند:

cargo build --release --features opencv
قابلیتتوضیحات
opencvفعال‌سازی bindingهای OpenCV برای پردازش تصویر و بینایی کامپیوتر
lunaفعال‌سازی ابزار Luna GUI برای Linux
pythonفعال‌سازی تعامل با Python برای فراخوانی مستقیم کتابخانه‌های Python

می‌توانید چندین قابلیت را ترکیب کنید:

cargo build --release --features "opencv,luna,python"

تأیید نصب

بررسی کنید که زمین نصب شده و کار می‌کند:

$ zamin version
zamin 1.7.3

REPL تعاملی را راه‌اندازی کنید تا از عملکرد زمان اجرا مطمئن شوید:

$ zamin repl
Zamin 1.7.3 (interactive)
Type "help" for more information.
zamin>

اولین اسکریپت شما

یک فایل به نام hello.zamin با محتوای زیر ایجاد کنید:

print("Hello, Zamin!")

آن را اجرا کنید:

$ zamin run hello.zamin
Hello, Zamin!

بیایید چیز جالب‌تری امتحان کنیم. محتوای hello.zamin را با کد زیر جایگزین کنید:

func greet(name) {
    print("Hello, {name}!");
}

func fibonacci(n) {
    if n <= 1 {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

greet("World");

for i in 0..10 {
    print("fib({i}) = {fibonacci(i)}");
}

استفاده از REPL

REPL زمین برای آزمایش قابلیت‌های زبان و تست تکه‌کدها عالی است:

$ zamin repl
Zamin 1.7.3 (interactive)
Type "help" for more information.
zamin> print("Hello!")
Hello!
zamin> 2 + 2
4
zamin> func add(a, b) { a + b }
zamin> add(10, 20)
30
zamin> [1, 2, 3].map(func(x) { x * 2 })
[2, 4, 6]
zamin> .exit

راه‌اندازی پروژه

برای برنامه‌های بزرگتر، از scaffold پروژه داخلی زمین استفاده کنید:

$ zamin new myproject
Created new project "myproject"
  src/main.zamin
  zamin.json

این دستور یک دایرکتوری پروژه با ساختار زیر ایجاد می‌کند:

myproject/
├── zamin.json
└── src/
    └── main.zamin

zamin.json

فایل zamin.json فراداده و تنظیمات ساخت پروژه شما را تعریف می‌کند:

{
  "name": "myproject",
  "version": "1.0.0",
  "entry": "src/main.zamin",
  "dependencies": [],
  "features": []
}

src/main.zamin

نقطه ورود برنامه شما:

print("Welcome to myproject!")

پروژه خود را بسازید و اجرا کنید:

$ cd myproject
$ zamin run
Welcome to myproject!

برای جزئیات بیشتر درباره مدیریت پروژه، به مرجع CLI پروژه مراجعه کنید.
