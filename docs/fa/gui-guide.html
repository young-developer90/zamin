Щ…ШұШ¬Ш№
CLI ЩҫШұЩҲЪҳЩҮ Ш№Щ…Щ„Ъ©ШұШҜ ШіЩҲШ§Щ„Ш§ШӘ Щ…ШӘШҜШ§ЩҲЩ„
Ш®Ш§ЩҶЩҮ > ШұШ§ЩҮЩҶЩ…Ш§ЫҢ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ GUI
ШҜШұ Ш§ЫҢЩҶ ШөЩҒШӯЩҮ
ЩҶЩ…Ш§ЫҢ Ъ©Щ„ЫҢ ШҙШұЩҲШ№ ШіШұЫҢШ№ Щ…ШұШ¬Ш№ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§ Leo (ЩҫЩҶШ¬ШұЩҮ) Frame Label Button Entry Щ…ШҜЫҢШұЫҢШӘ ЪҶЫҢШҜЩ…Ш§ЩҶ Ш№Щ…Щ„ЫҢШ§ШӘ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§ Щ…Ш«Ш§Щ„: Щ…Ш§ШҙЫҢЩҶ ШӯШіШ§ШЁ Щ…Ш«Ш§Щ„: ЩҲЫҢШұШ§ЫҢШҙЪҜШұ Щ…ШӘЩҶ ЫҢШ§ШҜШҜШ§ШҙШӘвҖҢЩҮШ§ЫҢ ЩҫЩ„ШӘЩҒШұЩ…

ШұШ§ЩҮЩҶЩ…Ш§ЫҢ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ GUI

ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШҜШіЪ©ШӘШ§Щҫ ШЁЩҲЩ…ЫҢ ШЁШ§ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ GUI ЪҶЩҶШҜЩҫЩ„ШӘЩҒШұЩ…ЫҢ ШІЩ…ЫҢЩҶ ШЁШіШ§ШІЫҢШҜ.

ЩҶЩ…Ш§ЫҢ Ъ©Щ„ЫҢ

ШІЩ…ЫҢЩҶ ШЁШ§ ШҜЩҲ Ш§ШЁШІШ§Шұ GUI Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ Ш§ШІ API ЫҢЪ©ШіШ§ЩҶЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ ШЁЩҶШ§ШЁШұШ§ЫҢЩҶ Ъ©ШҜ ЩҶЩҲШҙШӘЩҮ ШҙШҜЩҮ ШЁШұШ§ЫҢ ЫҢЪ©ЫҢ ШЁШҜЩҲЩҶ ШӘШәЫҢЫҢШұ ШұЩҲЫҢ ШҜЫҢЪҜШұЫҢ Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶШҜ:

  • Sol (ЩҲЫҢЩҶШҜЩҲШІ) — ЫҢЪ© Ш§ШЁШІШ§Шұ Щ…ШЁШӘЩҶЫҢ ШЁШұ Win32 Ъ©ЩҮ ШЁЩҮ ШөЩҲШұШӘ Щ…Ш§ЪҳЩҲЩ„ ШҜШ§Ш®Щ„ЫҢ ШҜШұ ЩҲЫҢЩҶШҜЩҲШІ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ЩҮЫҢЪҶ ЩҲШ§ШЁШіШӘЪҜЫҢ Ш®Ш§ШұШ¬ЫҢ ЩҶЫҢШ§ШІ ЩҶШҜШ§ШұШҜ.
  • Luna (Щ„ЫҢЩҶЩҲЪ©Ші) — ЫҢЪ© Ш§ШЁШІШ§Шұ Щ…ШЁШӘЩҶЫҢ ШЁШұ GTK4 Ъ©ЩҮ ШҜШұ ЩҫШҙШӘ ЩҫШұЪҶЩ… ЩҲЫҢЪҳЪҜЫҢ luna ШҜШұ ШҜШіШӘШұШі Ш§ШіШӘ. ЩҶЫҢШ§ШІ ШЁЩҮ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ ШӘЩҲШіШ№ЩҮ GTK4 ШҜШұ ШіЫҢШіШӘЩ… Щ…ЫҢШІШЁШ§ЩҶ ШҜШ§ШұШҜ.

ЩҮШұ ШҜЩҲ Ш§ШЁШІШ§Шұ ЩҶШ§Щ…вҖҢЩҮШ§ЫҢ ШӘШ§ШЁШ№ ЩҲ ШіШ§ШІЩҶШҜЩҮвҖҢЩҮШ§ЫҢ ЩҲЫҢШ¬ШӘ ЫҢЪ©ШіШ§ЩҶЫҢ ШұШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ. ШҙЩ…Ш§ Ш§ШЁШІШ§Шұ Щ…ШӘЩҶШ§ШіШЁ ШЁШ§ ЩҫЩ„ШӘЩҒШұЩ… Ш®ЩҲШҜ ШұШ§ Ш§ЩҶШӘШ®Ш§ШЁ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ ЩҲ ШЁЩӮЫҢЩҮ Ъ©ШҜ ШҙЩ…Ш§ ЫҢЪ©ШіШ§ЩҶ Щ…ЫҢвҖҢЩ…Ш§ЩҶШҜ.

💡 ШӘЩҲШ¬ЩҮ

ШҜШұ ЩҲЫҢЩҶШҜЩҲШІШҢ sol() ЩҮЩ…ЫҢШҙЩҮ ШҜШұ ШҜШіШӘШұШі Ш§ШіШӘ. ШҜШұ Щ„ЫҢЩҶЩҲЪ©ШіШҢ Ш§ШІ luna() Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ЩҲ ШЁШ§ --features luna ШЁШіШ§ШІЫҢШҜ.

ШҙШұЩҲШ№ ШіШұЫҢШ№

ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮ GUI ШӯШҜШ§ЩӮЩ„ЫҢ Ъ©ЩҮ ЫҢЪ© ЩҫЩҶШ¬ШұЩҮ ШЁШ§ ЫҢЪ© ШЁШұЪҶШіШЁ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ:

let gui = sol()  // or luna() on Linux
let win = gui.Leo("My App", 400, 300)
let label = gui.Label(win, "Hello, Zamin!")
gui.pack(label)
gui.mainloop(win)

Ш§ЫҢЩҶ ШұШ§ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ app.zamin Ш°Ш®ЫҢШұЩҮ Ъ©ЩҶЫҢШҜ ЩҲ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ:

$ zamin run app.zamin

ЫҢЪ© ЩҫЩҶШ¬ШұЩҮ ШЁШ§ Ш№ЩҶЩҲШ§ЩҶ "My App" ШЁШ§ Щ…ШӘЩҶ "Hello, Zamin!" ШҜШұ ЩҲШіШ· ШёШ§ЩҮШұ Щ…ЫҢвҖҢШҙЩҲШҜ.

Щ…ШұШ¬Ш№ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§

Leo (ЩҫЩҶШ¬ШұЩҮ)

ЩҫЩҶШ¬ШұЩҮ ШіШ·Шӯ ШЁШ§Щ„Ш§ Ъ©ЩҮ ЩҮЩ…ЩҮ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұ ШұШ§ ШҜШұ Ш®ЩҲШҜ ЩҶЪҜЩҮ Щ…ЫҢвҖҢШҜШ§ШұШҜ.

Ш§Щ…Ш¶Ш§ШӘЩҲШ¶ЫҢШӯШ§ШӘ
Leo(title?, width?, height?) ЫҢЪ© ЩҫЩҶШ¬ШұЩҮ ШіШ·Шӯ ШЁШ§Щ„Ш§ Ш¬ШҜЫҢШҜ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЪҜШұ Ш№ШұШ¶/Ш§ШұШӘЩҒШ§Ш№ ШӯШ°ЩҒ ШҙЩҲЩҶШҜШҢ ЩҫЫҢШҙвҖҢЩҒШұШ¶ 640×480 Ш§ШіШӘ. 
let gui = sol()
let win = gui.Leo("Settings", 800, 600)

Frame

ЫҢЪ© ЩҲЫҢШ¬ШӘ ШёШұЩҒ ШЁШұШ§ЫҢ ЪҜШұЩҲЩҮвҖҢШЁЩҶШҜЫҢ ЩҲ ШіШ§ШІЩ…Ш§ЩҶШҜЩҮЫҢ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§ЫҢ ЩҒШұШІЩҶШҜ.

Ш§Щ…Ш¶Ш§ШӘЩҲШ¶ЫҢШӯШ§ШӘ
Frame(parent) ЫҢЪ© ЩҒШұЫҢЩ… ШҜШұ ШҜШ§Ш®Щ„ ЩҲЫҢШ¬ШӘ ЩҲШ§Щ„ШҜ ШҜШ§ШҜЩҮ ШҙШҜЩҮ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. 
let gui = sol()
let win = gui.Leo("App", 400, 300)
let frame = gui.Frame(win)
gui.pack(frame)

Label

Щ…ШӘЩҶ Ш§ЫҢШіШӘШ§ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ. ЩҲШұЩҲШҜЫҢ Ъ©Ш§ШұШЁШұ ШұШ§ ЩҶЩ…ЫҢвҖҢЩҫШ°ЫҢШұШҜ.

Ш§Щ…Ш¶Ш§ШӘЩҲШ¶ЫҢШӯШ§ШӘ
Label(parent, text?) ЫҢЪ© ШЁШұЪҶШіШЁ ШЁШ§ Щ…ШӘЩҶ ШҜШ§ШҜЩҮ ШҙШҜЩҮ ШҜШұ ШҜШ§Ш®Щ„ ЩҲЫҢШ¬ШӘ ЩҲШ§Щ„ШҜ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. 
let gui = sol()
let win = gui.Leo("App", 400, 300)
let lbl = gui.Label(win, "Status: Ready")
gui.pack(lbl)

Button

ЫҢЪ© ШҜЪ©Щ…ЩҮ ЩӮШ§ШЁЩ„ Ъ©Щ„ЫҢЪ© Ъ©ЩҮ ШЁШ§ ЩҒШҙШ§Шұ ШҜШ§ШҜЩҶ ЫҢЪ© callback ШұШ§ ЩҒШ№Ш§Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Ш§Щ…Ш¶Ш§ШӘЩҲШ¶ЫҢШӯШ§ШӘ
Button(parent, text?, callback?) ЫҢЪ© ШҜЪ©Щ…ЩҮ ШЁШ§ ШЁШұЪҶШіШЁ ЩҲ handler Ъ©Щ„ЫҢ Ш§Ш®ШӘЫҢШ§ШұЫҢ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. 
let gui = sol()
let win = gui.Leo("App", 400, 300)

func on_click() {
    print("Button was clicked!");
}

let btn = gui.Button(win, "Click Me", on_click)
gui.pack(btn)
gui.mainloop(win)

CallbackЩҮШ§ ШӘЩҲШ§ШЁШ№ Щ…Ш№Щ…ЩҲЩ„ЫҢ ШІЩ…ЫҢЩҶ ЩҮШіШӘЩҶШҜ. Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШўШұЪҜЩҲЩ…Ш§ЩҶ ШЁЩҫШ°ЫҢШұЩҶШҜ ЫҢШ§ closureЩҮШ§ЫҢЫҢ ШЁШ§ШҙЩҶШҜ Ъ©ЩҮ ЩҲШ¶Ш№ЫҢШӘ Щ…ШӯЩ„ЫҢ ШұШ§ Ш¶ШЁШ· Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ.

Entry

ЫҢЪ© ЩҒЫҢЩ„ШҜ ЩҲШұЩҲШҜ Щ…ШӘЩҶ ШӘЪ©вҖҢШ®Ш·ЫҢ ШЁШұШ§ЫҢ ЩҲШұЩҲШҜЫҢ Ъ©Ш§ШұШЁШұ.

Ш§Щ…Ш¶Ш§ШӘЩҲШ¶ЫҢШӯШ§ШӘ
Entry(parent) ЫҢЪ© ЩҒЫҢЩ„ШҜ ЩҲШұЩҲШҜ Щ…ШӘЩҶ ШҜШұ ШҜШ§Ш®Щ„ ЩҲЫҢШ¬ШӘ ЩҲШ§Щ„ШҜ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. 
let gui = sol()
let win = gui.Leo("App", 400, 300)
let entry = gui.Entry(win)
gui.pack(entry)

Щ…ШҜЫҢШұЫҢШӘ ЪҶЫҢШҜЩ…Ш§ЩҶ

Ш§ШЁШІШ§ШұЩҮШ§ЫҢ GUI ШІЩ…ЫҢЩҶ ШҜЩҲ ШұЩҲШҙ ЪҶЫҢШҜЩ…Ш§ЩҶ ШЁШұШ§ЫҢ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ:

pack

Ш§Щ…Ш¶Ш§ШӘЩҲШ¶ЫҢШӯШ§ШӘ
pack(widget, side?, padx?, pady?) ЫҢЪ© ЩҲЫҢШ¬ШӘ ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…ШҜЫҢШұ ЩҮЩҶШҜШіЫҢ pack ЩӮШұШ§Шұ Щ…ЫҢвҖҢШҜЩҮШҜ. side Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ "top"ШҢ "bottom"ШҢ "left"ШҢ ЫҢШ§ "right" ШЁШ§ШҙШҜ. padx ЩҲ pady ШЁЩҮ ШӘШұШӘЫҢШЁ ЩҒШ§ШөЩ„ЩҮ Ш§ЩҒЩӮЫҢ ЩҲ Ш№Щ…ЩҲШҜЫҢ ШұШ§ ШЁШұ ШӯШіШЁ ЩҫЫҢЪ©ШіЩ„ Ш§Ш¶Ш§ЩҒЩҮ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. 
gui.pack(label, "top", 10, 5)
gui.pack(btn, "left", 5, 5)

place

Ш§Щ…Ш¶Ш§ШӘЩҲШ¶ЫҢШӯШ§ШӘ
place(widget, x, y, w?, h?) ЫҢЪ© ЩҲЫҢШ¬ШӘ ШұШ§ ШҜШұ Щ…Ш®ШӘШөШ§ШӘ Щ…Ш·Щ„ЩӮ ЩӮШұШ§Шұ Щ…ЫҢвҖҢШҜЩҮШҜ. w ЩҲ h Ш№ШұШ¶ ЩҲ Ш§ШұШӘЩҒШ§Ш№ ШұШ§ ШҜШұ ШөЩҲШұШӘ ШӘЩ…Ш§ЫҢЩ„ ШӘЩҶШёЫҢЩ… Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. 
gui.place(label, 20, 50, 200, 30)

Ш№Щ…Щ„ЫҢШ§ШӘ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§

Ш§ЫҢЩҶ ШӘЩҲШ§ШЁШ№ ЩҲЫҢШ¬ШӘвҖҢЩҮШ§ ШұШ§ ЩҫШі Ш§ШІ Ш§ЫҢШ¬Ш§ШҜ ШӘШәЫҢЫҢШұ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ ЫҢШ§ Ш§ШІ ШўЩҶЩҮШ§ ЩҫШұШівҖҢЩҲШ¬ЩҲ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ.

ШӘШ§ШЁШ№ШӘЩҲШ¶ЫҢШӯШ§ШӘ
config(widget, property, value) ШӘЩҶШёЫҢЩ… ЫҢЪ© ЩҲЫҢЪҳЪҜЫҢ ШұЩҲЫҢ ЩҲЫҢШ¬ШӘ. ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШұШ§ЫҢШ¬: "text"ШҢ "command"ШҢ "font"ШҢ "fg"ШҢ "bg".
get(entry) Щ…ШӯШӘЩҲШ§ЫҢ Щ…ШӘЩҶЫҢ Ш¬Ш§ШұЫҢ ЫҢЪ© ЩҲЫҢШ¬ШӘ Entry ШұШ§ ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜ.
insert(entry, pos, text) Щ…ШӘЩҶ ШұШ§ ШҜШұ Щ…ЩҲЩӮШ№ЫҢШӘ ШҜШ§ШҜЩҮ ШҙШҜЩҮ ШҜШұ ЫҢЪ© ЩҲЫҢШ¬ШӘ Entry ШҜШұШ¬ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
delete(entry, start, end?) Ъ©Ш§ШұШ§Ъ©ШӘШұЩҮШ§ ШұШ§ Ш§ШІ start ШӘШ§ end ШҜШұ ЫҢЪ© ЩҲЫҢШ¬ШӘ Entry ШӯШ°ЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЪҜШұ end ШӯШ°ЩҒ ШҙЩҲШҜШҢ ЫҢЪ© Ъ©Ш§ШұШ§Ъ©ШӘШұ ШӯШ°ЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

geometry(window, w, h) Ш§ЩҶШҜШ§ШІЩҮ ЫҢЪ© ЩҫЩҶШ¬ШұЩҮ ШұШ§ ШЁЩҮ Ш№ШұШ¶ ЩҲ Ш§ШұШӘЩҒШ§Ш№ ШҜШ§ШҜЩҮ ШҙШҜЩҮ ШӘШәЫҢЫҢШұ Щ…ЫҢвҖҢШҜЩҮШҜ.

Щ…Ш«Ш§Щ„: Щ…Ш§ШҙЫҢЩҶ ШӯШіШ§ШЁ

ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮ Щ…Ш§ШҙЫҢЩҶ ШӯШіШ§ШЁ Ъ©Ш§Щ…Щ„ ШЁШ§ ЩҒЫҢЩ„ШҜ ЩҶЩ…Ш§ЫҢШҙ ЩҲ ШҜЪ©Щ…ЩҮвҖҢЩҮШ§ЫҢ ШӯШіШ§ШЁЫҢ:

let gui = sol()
let win = gui.Leo("Calculator", 280, 350)

let display = gui.Entry(win)
gui.pack(display, "top", 10, 10)

let expression = ""

func on_digit(d) {
    expression = expression + d;
    gui.config(display, "text", expression);
}

func on_operator(op) {
    expression = expression + " " + op + " ";
    gui.config(display, "text", expression);
}

func on_clear() {
    expression = "";
    gui.config(display, "text", "");
}

func on_equal() {
    let result = eval(expression);
    gui.config(display, "text", str(result));
    expression = str(result);
}

// Build the button row for digits 1-9
for row in 0..3 {
    let frame = gui.Frame(win);
    gui.pack(frame, "top");
    for col in 0..3 {
        let digit = str(row * 3 + col + 1);
        let d = digit;
        let btn = gui.Button(frame, digit, func() { on_digit(d) });
        gui.pack(btn, "left", 2, 2);
    }
}

// Bottom row: 0, clear, equals
let bottom = gui.Frame(win)
gui.pack(bottom, "top")
let b0 = gui.Button(bottom, "0", func() { on_digit("0") })
gui.pack(b0, "left", 2, 2)
let bc = gui.Button(bottom, "C", on_clear)
gui.pack(bc, "left", 2, 2)
let be = gui.Button(bottom, "=", on_equal)
gui.pack(be, "left", 2, 2)

// Operator buttons
let ops = gui.Frame(win)
gui.pack(ops, "top")
for op in ["+", "-", "*", "/"] {
    let o = op;
    let b = gui.Button(ops, op, func() { on_operator(o) });
    gui.pack(b, "left", 2, 2);
}

gui.mainloop(win)

Щ…Ш«Ш§Щ„: ЩҲЫҢШұШ§ЫҢШҙЪҜШұ Щ…ШӘЩҶ

ЫҢЪ© ЩҲЫҢШұШ§ЫҢШҙЪҜШұ Щ…ШӘЩҶ ЩҫШ§ЫҢЩҮ ШЁШ§ ЩҒЫҢЩ„ШҜ ЩҲШұЩҲШҜЫҢШҢ ШҜЪ©Щ…ЩҮ ШЁШ§ШІ Ъ©ШұШҜЩҶ ЩҲ ШҜЪ©Щ…ЩҮ Ш°Ш®ЫҢШұЩҮ:

let gui = sol()
let win = gui.Leo("Text Editor", 600, 400)

// File path entry
let top_frame = gui.Frame(win)
gui.pack(top_frame, "top", 5, 5)

gui.Label(top_frame, "File:")
let path_entry = gui.Entry(top_frame)
gui.pack(path_entry, "left", 5, 5)

// Text area (simulated with a large Entry)
let text_area = gui.Entry(win)
gui.pack(text_area, "top", 10, 10)

// Button bar
let btn_frame = gui.Frame(win)
gui.pack(btn_frame, "top", 5, 5)

func on_open() {
    let path = gui.get(path_entry);
    if path == "" {
        gui.messagebox("Enter a file path first.", "Error");
        return;
    }
    let content = read(path);
    gui.config(text_area, "text", content);
    gui.title(win, "Text Editor - {path}");
}

func on_save() {
    let path = gui.get(path_entry);
    if path == "" {
        gui.messagebox("Enter a file path first.", "Error");
        return;
    }
    let content = gui.get(text_area);
    write(path, content);
    gui.messagebox("File saved!", "Success");
}

let open_btn = gui.Button(btn_frame, "Open", on_open)
gui.pack(open_btn, "left", 5, 5)

let save_btn = gui.Button(btn_frame, "Save", on_save)
gui.pack(save_btn, "left", 5, 5)

gui.mainloop(win)

Sol (ЩҲЫҢЩҶШҜЩҲШІ)

  • Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШЁШұ ШұЩҲЫҢ Win32 API ШіШ§Ш®ШӘЩҮ ШҙШҜЩҮ Ш§ШіШӘ — ШЁШҜЩҲЩҶ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ Ш®Ш§ШұШ¬ЫҢ ЫҢШ§ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§.
  • ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ© Щ…Ш§ЪҳЩҲЩ„ ШҜШ§Ш®Щ„ЫҢ ШҜШұ ШіШ§Ш®ШӘвҖҢЩҮШ§ЫҢ ЩҲЫҢЩҶШҜЩҲШІЫҢ ШІЩ…ЫҢЩҶ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
  • Ш§ШІ ШёШ§ЩҮШұ ЩҲ ШӯШі Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ЩҲЫҢЩҶШҜЩҲШІ ШЁШ§ ШұЩҶШҜШұ ШЁЩҲЩ…ЫҢ ЩҲЫҢШ¬ШӘ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Luna (Щ„ЫҢЩҶЩҲЪ©Ші)

  • ШЁШұ Ш§ШіШ§Ші GTK4 ШЁШұШ§ЫҢ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ ШҜШіЪ©ШӘШ§Щҫ Щ…ШҜШұЩҶ Щ„ЫҢЩҶЩҲЪ©Ші ШіШ§Ш®ШӘЩҮ ШҙШҜЩҮ Ш§ШіШӘ.
  • ЩҶЫҢШ§ШІЩ…ЩҶШҜ ЩҶШөШЁ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ ШӘЩҲШіШ№ЩҮ GTK4 Ш§ШіШӘ:
# Debian / Ubuntu
sudo apt install libgtk-4-dev

# Fedora
sudo dnf install gtk4-devel

# Arch
sudo pacman -S gtk4
  • ШІЩ…ЫҢЩҶ ШұШ§ ШЁШ§ ЩҲЫҢЪҳЪҜЫҢ luna ЩҒШ№Ш§Щ„ ШЁШіШ§ШІЫҢШҜ:
cargo build --release --features luna

macOS

  • ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ macOS ШЁШұШ§ЫҢ ЩҶШіШ®ЩҮ ШўЫҢЩҶШҜЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢШұЫҢШІЫҢ ШҙШҜЩҮ Ш§ШіШӘ. Ш§Щ…ШұЩҲШІ ШҜШұ macOS Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Щ…Ш§ЪҳЩҲЩ„вҖҢЩҮШ§ЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШЁШұШ§ЫҢ ШіШ§Ш®ШӘ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Ш®Ш· ЩҒШұЩ…Ш§ЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.
✔ ЩҶЪ©ШӘЩҮ

Ш§ШІ ШўЩҶШ¬Ш§ЫҢЫҢ Ъ©ЩҮ Sol ЩҲ Luna Ш§ШІ API ЫҢЪ©ШіШ§ЩҶЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ъ©ШҜ GUI Ш®ЩҲШҜ ШұШ§ ЫҢЪ© ШЁШ§Шұ ШЁЩҶЩҲЫҢШіЫҢШҜ ЩҲ ШЁШ§ Ш§ЩҶШӘШ®Ш§ШЁ ШіШ§ШІЩҶШҜЩҮ Ш§ШЁШІШ§Шұ Щ…ЩҶШ§ШіШЁ ШҜШұ ШЁШ§Щ„Ш§ЫҢ ЩҒШ§ЫҢЩ„ШҢ ШўЩҶ ШұШ§ ШЁШұШ§ЫҢ ЩҫЩ„ШӘЩҒШұЩ…вҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ Ъ©ЩҶЫҢШҜ.

ЩҲЫҢШұШ§ЫҢШҙ Ш§ЫҢЩҶ ШөЩҒШӯЩҮ ШҜШұ ЪҜЫҢШӘвҖҢЩҮШ§ШЁ ШўШ®ШұЫҢЩҶ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ: Ш¬ЩҲЩ„Ш§ЫҢ ЫІЫ°ЫІЫ¶
