پرسش‌ها و پاسخ‌های رایج درباره زبان برنامه‌نویسی زمین.

چگونه زمین را نصب کنم؟

زمین را از روی منبع بسازید (تنها روش نصب پشتیبانی‌شده):

git clone https://github.com/young-developer90/zamin.git
cd zamin
cargo build --release

فایل باینری در target/release/zamin قرار خواهد گرفت. آن را به PATH خود اضافه کنید.

چگونه رشته‌ها را به هم بچسبانم؟

از عملگر + استفاده کنید (هر دو عملوند باید رشته باشند) یا عملگر .. (غیررشته‌ها را به رشته تبدیل می‌کند).

let greeting = "Hello" + " " + "World";
print(greeting);  // "Hello World"

// Use .. to coerce non-strings to strings
let n = 42;
print("The answer is " .. n);  // "The answer is 42"

// f-strings are preferred for interpolation
let name = "Alice";
print("Hello {name}");

تفاوت let و const چیست؟

let یک متغیر قابل تغییر اعلام می‌کند. const یک ثابت غیرقابل تغییر اعلام می‌کند که باید در زمان کامپایل مقداردهی شود.

let x = 10
x = 20  // OK

const PI = 3.14159
PI = 3.0  // Error: cannot reassign constant

چگونه فایل‌ها را بخوانم؟

از ماژول کتابخانه استاندارد fs استفاده کنید:

let content = fs.read("data.txt");
print(content);

// Write to a file
fs.write("output.txt", "Hello, Zamin!");

چگونه درخواست HTTP بفرستم؟

از ماژول http استفاده کنید:

let resp = http.get("https://api.example.com/data");
print(resp.status);    // 200
print(resp.body);      // response body string

// POST with JSON body
let resp2 = http.post("https://api.example.com/submit",
    json.stringify({"name": "Alice", "email": "alice@example.com"})
);
print(resp2.status);

چگونه خطاها را مدیریت کنم؟

زمین از try/catch برای مدیریت خطا و throw برای ایجاد خطا استفاده می‌کند:

try {
    let content = fs.read("nonexistent.txt");
} catch (e) {
    print(f"Error: {e}");
}

// Throw errors explicitly
func validate(n) {
    if n < 0 {
        throw Error("negative value not allowed");
    }
    return n;
}

try {
    validate(-1);
} catch (e) {
    print(f"Caught: {e}");
}

چگونه از عبارت‌های منظم استفاده کنم؟

از ماژول re استفاده کنید:

let text = "Contact: user@example.com, admin@test.com";

let first = re.find(r"[\w.]+@[\w.]+", text);
print(first);                          // "user@example.com"

let all = re.find_all(r"[\w.]+@[\w.]+", text);
print(all);                            // ["user@example.com", "admin@test.com"]

// Check if pattern matches
print(re.is_match(r"\d+", "abc 123"));  // true

// Replace matches
let cleaned = re.sub(r"\d+", "#", "abc 123 def 456");
print(cleaned);                         // "abc # def 456"

// Extract capture groups
let caps = re.captures(r"(\w+)@(\w+)", "user@example.com");
print(caps);                            // ["user@example.com", "user", "example.com"]

چگونه JSON را تجزیه کنم؟

از ماژول json استفاده کنید:

let data = json.parse('{"name": "Alice", "age": 30}');
print(data["name"]);   // "Alice"
print(data["age"]);    // 30

// Serialize to JSON string
let obj = {"name": "Bob", "age": 25};
let str = json.stringify(obj);
print(str);             // {"name":"Bob","age":25}

// Pretty-print with indentation
print(json.pretty(obj));

چگونه یک پروژه ایجاد کنم؟

از دستور zamin new استفاده کنید:

zamin new my_project
cd my_project
zamin run src/main.zamin

این دستور ساختار استاندارد پروژه را با دایرکتوری‌های zamin.json، src/main.zamin، tests/ و modules/ ایجاد می‌کند.

چگونه تست‌ها را اجرا کنم؟

توابع تست را با ماژول test بنویسید، سپس zamin test را اجرا کنید:

use test

func test_addition() {
    test.assert_eq(1 + 1, 2);
}

func test_string_upper() {
    test.assert_eq("hello".upper(), "HELLO");
}
zamin test

فیلتر تست‌ها با نام: zamin test addition.

آیا زمین از چندنخی پشتیبانی می‌کند؟

زمین از یک ماشین مجازی تک‌رشته‌ای با همزمانی مشارکتی از طریق فیبر‌ها (coroutine‌های سبک) استفاده می‌کند. هر فیبر در همان رشته اجرا می‌شود و کنترل را به صراحت واگذار می‌کند و امکان زمان‌بندی وظایف همزمان بدون مشکلات همگام‌سازی حالت مشترک را فراهم می‌کند.

برای کارهای محدود به CPU، زمین می‌تواند فرآیندهای مستقل را از طریق ماژول subprocess ایجاد کند. موازی‌سازی واقعی در چندین رشته OS از طریق API جاسازی Rust هنگام فراخوانی زمین از یک برنامه Rust چندرشته‌ای در دسترس است.

چگونه از GUI در Linux استفاده کنم؟

زمین شامل یک ابزار GUI داخلی است که روی Linux از طریق GTK4 کار می‌کند. وابستگی‌های سیستم مورد نیاز را نصب کنید:

# Ubuntu/Debian
sudo apt install libgtk-4-dev

# Fedora
sudo dnf install gtk4-devel

سپس از ماژول luna استفاده کنید (با --features luna بسازید):

let win = luna.Leo("My App", 800, 600);
let label = luna.Label(win, "Hello, Linux!");
luna.pack(label);
luna.mainloop(win);

برای جزئیات کامل به راهنمای GUI مراجعه کنید.

چگونه از OpenCV استفاده کنم؟

زمین OpenCV را از طریق ماژول cv پوشش می‌دهد. ابتدا OpenCV را روی سیستم خود نصب کنید، سپس:

use cv

let img = cv.imread("photo.jpg")
let gray = cv.cvtColor(img, cv.COLOR_BGR2GRAY)
let edges = cv.canny(gray, 50, 150)
cv.imwrite("edges.jpg", edges)
print("Saved edges.jpg")

برای جزئیات کامل به راهنمای OpenCV مراجعه کنید.

چرا برنامه من کند است؟

علل و رفع‌های رایج:

  • الحاق رشته در حلقه‌ها — به جای + مکرر روی رشته‌ها از f-strings یا list.join() استفاده کنید.
  • دسترسی به متغیر سراسری — متغیرهای محلی حدود ۱۰ برابر سریع‌تر هستند. در حلقه‌های فشرده، متغیرهای سراسری را در متغیرهای محلی کش کنید.
  • تخصیص مکرر لیست — از list.new(capacity) برای پیش‌تخصیص استفاده کنید.
  • ساخت بهینه‌نشده — برای ساخت‌های release با بهینه‌سازی‌های AOT از zamin build استفاده کنید.
  • الگوریتم نابهینه — ابتدا با zamin run --disassemble پروفایل کنید تا بایت‌کد بررسی شود، سپس بهبودهای الگوریتمی را در نظر بگیرید.

برای بنچمارک‌های دقیق و تکنیک‌های بهینه‌سازی به صفحه عملکرد مراجعه کنید.

چگونه می‌توانم در زمین مشارکت کنم؟

زمین متن‌باز با مجوز MIT است. برای مشارکت:

  • گزارش باگ — در github.com/young-developer90/zamin/issues issue باز کنید
  • ارسال کد — مخزن را fork کنید، یک شاخه ویژگی ایجاد کنید و pull request باز کنید
  • بهبود مستندات — تمام مستندات در دایرکتوری docs/ به صورت فایل‌های HTML قرار دارند
  • اشتراک بازخورد — درخواست‌های ویژگی و پیشنهادات در issue tracker خوش‌آمد هستند

برای دستورالعمل‌های ساخت و ساختار پروژه به README مراجعه کنید.

چگونه زمین را در برنامه Rust خود جاسازی کنم؟

زمین در crates.io منتشر شده است. آن را به عنوان وابستگی اضافه کنید و از API جاسازی استفاده کنید:

// Cargo.toml
[dependencies]
zamin = "1.7.3"
// main.rs
func main() {
    zamin::execute_source(r#"
        print("Hello from embedded Zamin!");
    "#).unwrap();

    zamin::execute_file("scripts/plugin.zamin").unwrap();
}

برای مرجع کامل API به بخش جاسازی در Rust مراجعه کنید.
