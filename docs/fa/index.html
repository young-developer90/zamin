زمین

یک زبان برنامه‌نویسی سریع و خواناتا

زمین ترکیبی از نحو شبیه به Python با یک ماشین مجازی بایت‌کد سبک است. این زبان در حلقه‌های سنگین عددی با Python رقابت می‌کند و با بیش از ۲۵ ماژول کتابخانه استاندارد داخلی، یک ابزار GUI بومی و قابلیت تعامل با C و Python عرضه می‌شود — آماده برای پروژه‌های واقعی.

⚡

ماشین مجازی بایت‌کد

ماشین مجازی بایت‌کد مبتنی بر ثبات با dispatch خام، opcodeهای تخصصی اعداد صحیح و بهینه‌ساز نگاه‌به‌جلو. در حلقه‌های عددی با CPython رقابت می‌کند و بیش از ۲۵ ماژول داخلی در دسترس شماست.

📝

نحو خواناتا

نحوی شبیه به Python که از همان ابتدا طبیعی به نظر می‌رسد. بلوک‌های مبتنی بر تورفتگی، عبارات تمیز و حاشیه‌نویسی نوع ساده، کد شما را خواناتا و قابل نگهداری می‌کند.

📦

خودکفا

بیش از ۲۵ ماژول کتابخانه استاندارد به صورت پیش‌فرض ارائه می‌شود. ورودی/خروجی فایل، شبکه، JSON، regex، ریاضی، تاریخ و زمان و موارد دیگر — بدون نیاز به مدیر بسته خارجی برای کارهای روزمره.

🌐

چند پلتفرم

با یک پایگاه کد واحد روی Linux، macOS و Windows بسازید و اجرا کنید. کامپایلر و زمان اجرای زمین کاملاً قابل حمل هستند، بنابراین پروژه‌هایتان هر جا که می‌روید همراه شما هستند.

🔌

قابل گسترش

برای کدهای حساس به عملکرد، افزونه‌های C بنویسید، مستقیماً کتابخانه‌های Python را فراخوانی کنید یا از NVIDIA CUDA برای محاسبات شتاب‌یافته GPU بهره ببرید. زمین با اکوسیستمی که از قبل می‌شناسید به خوبی کار می‌کند.

🖼

GUI داخلی

با ابزار Sol GUI در Windows یا Luna در Linux، برنامه‌های دسکتاپ بومی ایجاد کنید. ویجت‌ها، چیدمان‌ها، مدیریت رویداد و استایل‌دهی همه بخشی از تجربه استاندارد هستند.

مثال سریع

در ادامه یک نمونه از کد زمین را مشاهده می‌کنید:

func greet(name) {
    print("Hello, {name}!");
}

func fibonacci(n) {
    if n <= 1 {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

greet("World");

for i in 0..10 {
    print("fib({i}) = {fibonacci(i)}");
}
