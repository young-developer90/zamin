راهنمای OpenCV

پردازش تصویر و بینایی کامپیوتر با OpenCV 4/5.

نمای کلی

ماژول opencv از OpenCV 4/5 استفاده می‌کند تا به برنامه‌های زمین دسترسی به ورودی/خروجی تصویر، تبدیل رنگ، فیلتر کردن، تبدیل‌ها، تشخیص اشیا، رسم و موارد دیگر بدهد. این ماژول در پشت یک پرچم ویژگی در دسترس است و نیاز به کتابخانه‌های توسعه OpenCV در سطح سیستم دارد.

با ویژگی opencv فعال بسازید:

cargo build --release --features opencv

کتابخانه‌های توسعه OpenCV را روی سیستم خود نصب کنید:

# Debian / Ubuntu
sudo apt install libopencv-dev

# Fedora
sudo dnf install opencv-devel

# macOS (Homebrew)
brew install opencv

شروع سریع

یک تصویر را بخوانید، به مقیاس خاکستری تبدیل کنید، تشخیص لبه اعمال کنید و نتیجه را ذخیره کنید:

let img = opencv.imread("photo.jpg")
let gray = opencv.grayscale(img)
let edges = opencv.canny(gray, 50.0, 150.0)
opencv.imwrite("edges.png", edges)

اجرا کنید:

$ zamin run detect.zamin

ورودی/خروجی تصویر

تابعتوضیحات
imread(path) خواندن یک تصویر از دیسک. یک ماتریس تصویر BGR برمی‌گرداند.
imwrite(path, image) نوشتن یک تصویر روی دیسک. فرمت از پسوند تشخیص داده می‌شود.
imread_gray(path) خواندن یک تصویر مستقیماً به صورت مقیاس خاکستری.
shape(image) [rows, cols, channels] را برای تصویر برمی‌گرداند.
copy(image) یک کپی عمیق از ماتریس تصویر برمی‌گرداند. 
let img = opencv.imread("photo.jpg")
print(opencv.shape(img))  // [480, 640, 3]

let clone = opencv.copy(img)
opencv.imwrite("copy.jpg", clone)

تبدیل رنگ

تابعتوضیحات
cvt_color(image, code) تبدیل فضای رنگ با استفاده از یک ثابت کد (مثلاً opencv.BGR2GRAY).
grayscale(image) تشخیص خودکار تعداد کانال‌ها و تبدیل به مقیاس خاکستری تک‌کاناله. 
let img = opencv.imread("photo.jpg")
let gray = opencv.grayscale(img)
let rgb = opencv.cvt_color(img, opencv.COLOR_BGR2RGB)

تغییر اندازه و تبدیل

تابعتوضیحات
resize(image, width, height) تغییر اندازه با استفاده از درون‌یابی خطی.
resize_fast(image, width, height) تغییر اندازه با استفاده از درون‌یابی نزدیک‌ترین همسایه (سریع‌تر، کیفیت کمتر).
thumbnail(image, max_size) تغییر اندازه با حفظ نسبت ابعاد به طوری که طولانی‌ترین ضلع برابر max_size شود.
flip(image, code) برگرداندن تصویر. از FLIP_HORIZONTAL، FLIP_VERTICAL، یا FLIP_BOTH استفاده کنید.
rotate(image, code) چرخش ۹۰ درجه. از ROTATE_90_CLOCKWISE، ROTATE_180، یا ROTATE_90_COUNTERCLOCKWISE استفاده کنید.
crop(image, x, y, w, h) برش یک ناحیه مستطیلی از پیکسل (x, y).
letterbox(image, target_w, target_h, bg_r?, bg_g?, bg_b?) تغییر اندازه و padding برای پر کردن ابعاد هدف، با افزودن رنگ پس‌زمینه (پیش‌فرض سیاه). 
let img = opencv.imread("photo.jpg")
let small = opencv.resize(img, 320, 240)
let thumb = opencv.thumbnail(img, 128)
let flipped = opencv.flip(img, opencv.FLIP_HORIZONTAL)
let rotated = opencv.rotate(img, opencv.ROTATE_90_CLOCKWISE)
let cropped = opencv.crop(img, 100, 50, 200, 200)
let padded = opencv.letterbox(img, 640, 640, 114, 114, 114)

فیلترها و افکت‌ها

تابعتوضیحات
gaussian_blur(image, kx, ky, sigma) اعمال محوکنندگی گاوسی با اندازه هسته (kx, ky) و سیگما.
blur(image, ksize) اعمال یک محوکنندگی جعبه‌ای ساده با اندازه هسته داده شده.
canny(image, low, high) تشخیص لبه Canny با آستانه پایین و بالا.
edges(image) تشخیص لبه Canny با آستانه‌های محاسبه‌شده خودکار.
threshold(image, thresh, maxval, type) اعمال آستانه‌گذاری دودویی. از ثابت‌هایی مانند THRESH_BINARY استفاده کنید.
brightness(image, delta) تنظیم روشنایی با افزودن delta به هر پیکسل.
contrast(image, alpha) تنظیم کنتراست. alpha > 1.0 افزایش، alpha < 1.0 کاهش.
invert(image) معکوس کردن همه مقادیر پیکسل (تصویر منفی).
sepia(image) اعمال افکت رنگ سپیا.
pixelate(image, block) پیکسل‌سازی تصویر با استفاده از بلوک‌های اندازه داده شده.
equalize_hist(image) هم‌سازی هیستوگرام برای بهبود کنتراست در تصاویر مقیاس خاکستری.
normalize(image, alpha?, beta?, norm_type?) نرمال‌سازی مقادیر پیکسل. پیش‌فرض محدوده 0-1 با نرم L2. 
let img = opencv.imread("photo.jpg")
let blurred = opencv.gaussian_blur(img, 15, 15, 0)
let edges = opencv.edges(img)
let bright = opencv.brightness(img, 30)
let dark = opencv.contrast(img, 0.5)
let inv = opencv.invert(img)
let sep = opencv.sepia(img)
let pix = opencv.pixelate(img, 8)

تشخیص اشیا

تابعتوضیحات
detect_cascade(image, cascade_path, ...) تشخیص اشیا با استفاده از یک فایل XML طبقه‌بند Haar cascade. یک لیست از مستطیل‌های مرزی [x, y, w, h] برمی‌گرداند.
detect_people(image, ...) تشخیص افراد با استفاده از آشکارساز عابر HOG. مستطیل‌های مرزی برمی‌گرداند.
detect_dnn(image, model_path, ...) تشخیص اشیا با استفاده از یک مدل DNN (مثلاً YOLO، MobileNet). مستطیل‌های مرزی با نمرات اطمینان برمی‌گرداند. 
let img = opencv.imread("street.jpg")

// Haar cascade face detection
let faces = opencv.detect_cascade(img, "haarcascade_frontalface.xml")
for face in faces {
    opencv.draw_rect(img, face[0], face[1], face[2], face[3], 0, 255, 0, 2);
    print("Face at: {face}");
}

// HOG people detection
let people = opencv.detect_people(img)
for p in people {
    opencv.draw_rect(img, p[0], p[1], p[2], p[3], 255, 0, 0, 2);
}

opencv.imwrite("detected.jpg", img)

رسم و حاشیه‌نویسی

تابعتوضیحات
draw_rect(image, x, y, w, h, r?, g?, b?, thickness?) رسم یک مستطیل. پیش‌فرض: رنگ سبز، ضخامت ۱.
draw_text(image, text, x, y, scale?, r?, g?, b?, thickness?) رسم متن در موقعیت (x, y). پیش‌فرض: scale 1.0، رنگ سفید، ضخامت ۱. 
let img = opencv.imread("photo.jpg")
opencv.draw_rect(img, 50, 50, 200, 150, 0, 255, 0, 3)
opencv.draw_text(img, "Detected!", 60, 45, 1.2, 0, 255, 0, 2)
opencv.imwrite("annotated.jpg", img)

تطبیق الگو

تابعتوضیحات
match_template(image, template, method?) جستجوی یک الگو درون یک تصویر. مکان بهترین تطبیق و اطمینان را برمی‌گرداند. روش پیش‌فرض TM_CCOEFF_NORMED است. 
let img = opencv.imread("screenshot.png")
let icon = opencv.imread("icon.png")
let result = opencv.match_template(img, icon)
print("Best match at: {result.location}, score: {result.score}")

نمایش (GUI)

برای نمایش سریع، ماژول opencv توابع نمایشی را ارائه می‌دهد که از ماژول highgui OpenCV استفاده می‌کنند.

تابعتوضیحات
imshow(name, image) نمایش یک تصویر در یک پنجره نام‌گذاری شده.
wait_key(delay) منتظر فشار کلید به مدت delay میلی‌ثانیه. برای انتظار نامحدود 0 را وارد کنید.
destroy_all_windows() بستن همه پنجره‌های نمایش باز. 
let img = opencv.imread("photo.jpg")
opencv.imshow("Preview", img)
opencv.wait_key(0)
opencv.destroy_all_windows()

ASCII Art

تبدیل تصاویر به ASCII art برای خروجی ترمینال یا افکت‌های خلاقانه.

تابعتوضیحات
to_ascii(image, width?) تبدیل یک تصویر به ASCII art. width اختیاری عرض خروجی را کنترل می‌کند (پیش‌فرض ۸۰).
read_ascii(path, width?) خواندن یک تصویر از دیسک و تبدیل به ASCII در یک مرحله. 
let img = opencv.imread("photo.jpg")
let art = opencv.to_ascii(img, 100)
print(art)

// Or in one step:
let art2 = opencv.read_ascii("photo.jpg", 80)
print(art2)

مثال کامل

یک مثال کامل که یک تصویر مصنوعی ایجاد می‌کند، چندین عملیات اعمال می‌کند، نتایج را ذخیره می‌کند و ASCII art چاپ می‌کند:

let opencv = require("opencv")

// Create a synthetic 400x300 image with color gradient
print("Creating synthetic image...")
let img = opencv.zeros(300, 400, 3)
for y in 0..300 {
    for x in 0..400 {
        let r = (float(x) / 400.0 * 255.0).int();
        let g = (float(y) / 300.0 * 255.0).int();
        img.set(y, x, [r, g, 128]);
    }
}

opencv.imwrite("gradient.png", img)
print("Saved gradient.png")

// Grayscale conversion
let gray = opencv.grayscale(img)
opencv.imwrite("gradient_gray.png", gray)
print("Saved gradient_gray.png")

// Gaussian blur
let blurred = opencv.gaussian_blur(img, 21, 21, 0)
opencv.imwrite("gradient_blur.png", blurred)
print("Saved gradient_blur.png")

// Edge detection
let edges = opencv.canny(gray, 50.0, 150.0)
opencv.imwrite("gradient_edges.png", edges)
print("Saved gradient_edges.png")

// Threshold
let thresh = opencv.threshold(gray, 128.0, 255.0, opencv.THRESH_BINARY)
opencv.imwrite("gradient_thresh.png", thresh)
print("Saved gradient_thresh.png")

// Sepia effect
let sep = opencv.sepia(img)
opencv.imwrite("gradient_sepia.png", sep)
print("Saved gradient_sepia.png")

// Pixelate
let pix = opencv.pixelate(img, 16)
opencv.imwrite("gradient_pixel.png", pix)
print("Saved gradient_pixel.png")

// Flip and rotate
let flipped = opencv.flip(img, opencv.FLIP_BOTH)
opencv.imwrite("gradient_flipped.png", flipped)
print("Saved gradient_flipped.png")

// Draw annotations
let annotated = opencv.copy(img)
opencv.draw_rect(annotated, 50, 50, 300, 200, 0, 255, 0, 3)
opencv.draw_text(annotated, "Zamin OpenCV", 80, 140, 2.0, 255, 255, 255, 3)
opencv.imwrite("gradient_annotated.png", annotated)
print("Saved gradient_annotated.png")

// Print ASCII art
print("\nASCII art preview:")
let art = opencv.to_ascii(gray, 60)
print(art)

// Print image info
print("\nImage shape: {opencv.shape(img)}")
print("Done!")

مرجع ثابت‌ها

کدهای تبدیل رنگ

ثابتتوضیحات
opencv.BGR2GRAYتبدیل BGR به مقیاس خاکستری
opencv.COLOR_BGR2RGBتبدیل BGR به RGB
opencv.COLOR_RGB2BGRتبدیل RGB به BGR
opencv.COLOR_BGR2HSVتبدیل BGR به HSV
opencv.COLOR_BGR2LABتبدیل BGR به CIE Lab
opencv.COLOR_GRAY2BGRتبدیل مقیاس خاکستری به BGR سه‌کاناله

انواع آستانه

ثابتتوضیحات
opencv.THRESH_BINARYپیکسل‌های > آستانه به maxval تنظیم می‌شوند، در غیر این صورت ۰
opencv.THRESH_BINARY_INVمعکوس THRESH_BINARY
opencv.THRESH_TRUNCپیکسل‌های > آستانه به آستانه تنظیم می‌شوند، در غیر این صورت بدون تغییر
opencv.THRESH_TOZEROپیکسل‌های > آستانه بدون تغییر، در غیر این صورت به ۰ تنظیم می‌شوند
opencv.THRESH_TOZERO_INVپیکسل‌های > آستانه به ۰ تنظیم می‌شوند، در غیر این صورت بدون تغییر

کدهای برگرداندن

ثابتتوضیحات
opencv.FLIP_HORIZONTALبرگرداندن چپ-به-راست (کد ۱)
opencv.FLIP_VERTICALبرگرداندن بالا-به-پایین (کد ۰)
opencv.FLIP_BOTHبرگرداندن هر دو محور (کد ۱-)

کدهای چرخش

ثابتتوضیحات

opencv.ROTATE_180چرخش ۱۸۰ درجه
opencv.ROTATE_90_COUNTERCLOCKWISEچرخش ۹۰ درجه در خلاف جهت عقربه‌های ساعت
