OpenCV ガイド

OpenCV 4/5 を搭載した画像処理とコンピュータビジョン。

概要

opencv モジュールは OpenCV 4/5 をラップし、Zamin プログラムに画像 I/O、色変換、フィルタリング、変換、物体検出、描画などの機能を提供します。フィーチャーフラグの背後で利用可能で、システムレベルの OpenCV 開発ライブラリが必要です。

opencv フィーチャーを有効にしてビルドします。

cargo build --release --features opencv

システムに OpenCV 開発ライブラリをインストールします。

# Debian / Ubuntu
sudo apt install libopencv-dev

# Fedora
sudo dnf install opencv-devel

# macOS (Homebrew)
brew install opencv

クイックスタート

画像を読み取り、グレースケールに変換し、エッジ検出を適用して結果を保存します。

let img = opencv.imread("photo.jpg")
let gray = opencv.grayscale(img)
let edges = opencv.canny(gray, 50.0, 150.0)
opencv.imwrite("edges.png", edges)

実行します。

$ zamin run detect.zamin

画像 I/O

関数説明
imread(path) ディスクから画像を読み取ります。BGR 画像行列を返します。
imwrite(path, image) 画像をディスクに書き込みます。形式は拡張子から推測されます。
imread_gray(path) 画像を直接グレースケールとして読み取ります。
shape(image) 画像の [rows, cols, channels] を返します。
copy(image) 画像行列のディープコピーを返します。 
let img = opencv.imread("photo.jpg")
print(opencv.shape(img))  // [480, 640, 3]

let clone = opencv.copy(img)
opencv.imwrite("copy.jpg", clone)

色変換

関数説明
cvt_color(image, code) コード定数を使用して色空間を変換します（例：opencv.BGR2GRAY）。
grayscale(image) チャンネル数を自動検出し、シングルチャンネルのグレースケールに変換します。 
let img = opencv.imread("photo.jpg")
let gray = opencv.grayscale(img)
let rgb = opencv.cvt_color(img, opencv.COLOR_BGR2RGB)

リサイズと変換

関数説明
resize(image, width, height) 線形補間を使用してリサイズします。
resize_fast(image, width, height) 最近傍補間を使用してリサイズします（高速、低品質）。
thumbnail(image, max_size) アスペクト比を維持してリサイズし、最長辺が max_size になります。
flip(image, code) 画像を反転します。FLIP_HORIZONTAL、FLIP_VERTICAL、FLIP_BOTH を使用します。
rotate(image, code) 90° 単位で回転します。ROTATE_90_CLOCKWISE、ROTATE_180、ROTATE_90_COUNTERCLOCKWISE を使用します。
crop(image, x, y, w, h) ピクセル (x, y) から始まる矩形領域を切り抜きます。
letterbox(image, target_w, target_h, bg_r?, bg_g?, bg_b?) ターゲットサイズに収まるようリサイズとパディングを行い、背景色を追加します（デフォルトは黒）。 
let img = opencv.imread("photo.jpg")
let small = opencv.resize(img, 320, 240)
let thumb = opencv.thumbnail(img, 128)
let flipped = opencv.flip(img, opencv.FLIP_HORIZONTAL)
let rotated = opencv.rotate(img, opencv.ROTATE_90_CLOCKWISE)
let cropped = opencv.crop(img, 100, 50, 200, 200)
let padded = opencv.letterbox(img, 640, 640, 114, 114, 114)

フィルタとエフェクト

関数説明
gaussian_blur(image, kx, ky, sigma) カーネルサイズ (kx, ky) と sigma でガウシアンブラーを適用します。
blur(image, ksize) 指定されたカーネルサイズで単純なボックスブラーを適用します。
canny(image, low, high) 低しきい値と高しきい値で Canny エッジ検出を行います。
edges(image) 自動計算されたしきい値で Canny エッジ検出を行います。
threshold(image, thresh, maxval, type) 二値しきい値処理を適用します。THRESH_BINARY などの定数を使用します。
brightness(image, delta) すべてのピクセルに delta を加算して明るさを調整します。
contrast(image, alpha) コントラストを調整します。alpha > 1.0 で増加、alpha < 1.0 で減少。
invert(image) すべてのピクセル値を反転します（ネガティブ画像）。
sepia(image) セピア調エフェクトを適用します。
pixelate(image, block) 指定されたサイズのブロックで画像をピクセル化します。
equalize_hist(image) ヒストグラムを均等化してグレースケール画像のコントラストを改善します。
normalize(image, alpha?, beta?, norm_type?) ピクセル値を正規化します。デフォルトは L2 ノルムで 0-1 範囲。 
let img = opencv.imread("photo.jpg")
let blurred = opencv.gaussian_blur(img, 15, 15, 0)
let edges = opencv.edges(img)
let bright = opencv.brightness(img, 30)
let dark = opencv.contrast(img, 0.5)
let inv = opencv.invert(img)
let sep = opencv.sepia(img)
let pix = opencv.pixelate(img, 8)

物体検出

関数説明
detect_cascade(image, cascade_path, ...) Haar カスケード分類器 XML ファイルを使用して物体を検出します。[x, y, w, h] の矩形リストを返します。
detect_people(image, ...) HOG 人物検出器を使用して人物を検出します。矩形を返します。
detect_dnn(image, model_path, ...) DNN モデル（YOLO、MobileNet など）を使用して物体を検出します。信頼度スコア付きの矩形を返します。 
let img = opencv.imread("street.jpg")

// Haar cascade face detection
let faces = opencv.detect_cascade(img, "haarcascade_frontalface.xml")
for face in faces {
    opencv.draw_rect(img, face[0], face[1], face[2], face[3], 0, 255, 0, 2);
    print("Face at: {face}");
}

// HOG people detection
let people = opencv.detect_people(img)
for p in people {
    opencv.draw_rect(img, p[0], p[1], p[2], p[3], 255, 0, 0, 2);
}

opencv.imwrite("detected.jpg", img)

描画とアノテーション

関数説明
draw_rect(image, x, y, w, h, r?, g?, b?, thickness?) 矩形を描画します。デフォルト：緑色、太さ 1。
draw_text(image, text, x, y, scale?, r?, g?, b?, thickness?) 位置 (x, y) にテキストを描画します。デフォルト：scale 1.0、白色、太さ 1。 
let img = opencv.imread("photo.jpg")
opencv.draw_rect(img, 50, 50, 200, 150, 0, 255, 0, 3)
opencv.draw_text(img, "Detected!", 60, 45, 1.2, 0, 255, 0, 2)
opencv.imwrite("annotated.jpg", img)

テンプレートマッチング

関数説明
match_template(image, template, method?) 画像内からテンプレートを検索します。最良一致の位置と信頼度を返します。メソッドのデフォルトは TM_CCOEFF_NORMED。 
let img = opencv.imread("screenshot.png")
let icon = opencv.imread("icon.png")
let result = opencv.match_template(img, icon)
print("Best match at: {result.location}, score: {result.score}")

表示（GUI）

簡単な可視化のために、opencv モジュールは OpenCV の highgui モジュールを使用した表示機能を提供します。

関数説明
imshow(name, image) 名前付きウィンドウに画像を表示します。
wait_key(delay) delay ミリ秒間キー入力を待機します。0 を渡すと無期限に待機します。
destroy_all_windows() 開いているすべての表示ウィンドウを閉じます。 
let img = opencv.imread("photo.jpg")
opencv.imshow("Preview", img)
opencv.wait_key(0)
opencv.destroy_all_windows()

ASCII アート

画像を ASCII アートに変換して、ターミナル出力やクリエイティブなエフェクトに使用できます。

関数説明
to_ascii(image, width?) 画像を ASCII アートに変換します。オプションの width で出力幅を制御します（デフォルト 80）。
read_ascii(path, width?) ディスクから画像を読み取り、1 ステップで ASCII に変換します。 
let img = opencv.imread("photo.jpg")
let art = opencv.to_ascii(img, 100)
print(art)

// Or in one step:
let art2 = opencv.read_ascii("photo.jpg", 80)
print(art2)

完全な例

合成画像を作成し、複数の操作を適用し、結果を保存して ASCII アートを表示する完全な動作例。

let opencv = require("opencv")

// Create a synthetic 400x300 image with color gradient
print("Creating synthetic image...")
let img = opencv.zeros(300, 400, 3)
for y in 0..300 {
    for x in 0..400 {
        let r = (float(x) / 400.0 * 255.0).int();
        let g = (float(y) / 300.0 * 255.0).int();
        img.set(y, x, [r, g, 128]);
    }
}

opencv.imwrite("gradient.png", img)
print("Saved gradient.png")

// Grayscale conversion
let gray = opencv.grayscale(img)
opencv.imwrite("gradient_gray.png", gray)
print("Saved gradient_gray.png")

// Gaussian blur
let blurred = opencv.gaussian_blur(img, 21, 21, 0)
opencv.imwrite("gradient_blur.png", blurred)
print("Saved gradient_blur.png")

// Edge detection
let edges = opencv.canny(gray, 50.0, 150.0)
opencv.imwrite("gradient_edges.png", edges)
print("Saved gradient_edges.png")

// Threshold
let thresh = opencv.threshold(gray, 128.0, 255.0, opencv.THRESH_BINARY)
opencv.imwrite("gradient_thresh.png", thresh)
print("Saved gradient_thresh.png")

// Sepia effect
let sep = opencv.sepia(img)
opencv.imwrite("gradient_sepia.png", sep)
print("Saved gradient_sepia.png")

// Pixelate
let pix = opencv.pixelate(img, 16)
opencv.imwrite("gradient_pixel.png", pix)
print("Saved gradient_pixel.png")

// Flip and rotate
let flipped = opencv.flip(img, opencv.FLIP_BOTH)
opencv.imwrite("gradient_flipped.png", flipped)
print("Saved gradient_flipped.png")

// Draw annotations
let annotated = opencv.copy(img)
opencv.draw_rect(annotated, 50, 50, 300, 200, 0, 255, 0, 3)
opencv.draw_text(annotated, "Zamin OpenCV", 80, 140, 2.0, 255, 255, 255, 3)
opencv.imwrite("gradient_annotated.png", annotated)
print("Saved gradient_annotated.png")

// Print ASCII art
print("\nASCII art preview:")
let art = opencv.to_ascii(gray, 60)
print(art)

// Print image info
print("\nImage shape: {opencv.shape(img)}")
print("Done!")

定数リファレンス

色変換コード

定数説明
opencv.BGR2GRAYBGR をグレースケールに変換
opencv.COLOR_BGR2RGBBGR を RGB に変換
opencv.COLOR_RGB2BGRRGB を BGR に変換
opencv.COLOR_BGR2HSVBGR を HSV に変換
opencv.COLOR_BGR2LABBGR を CIE Lab に変換
opencv.COLOR_GRAY2BGRグレースケールを 3 チャンネル BGR に変換

しきい値タイプ

定数説明
opencv.THRESH_BINARYしきい値 > のピクセルは maxval、それ以外は 0
opencv.THRESH_BINARY_INVTHRESH_BINARY の反転
opencv.THRESH_TRUNCしきい値 > のピクセルはしきい値に、それ以外はそのまま
opencv.THRESH_TOZEROしきい値 > のピクセルはそのまま、それ以外は 0
opencv.THRESH_TOZERO_INVしきい値 > のピクセルは 0、それ以外はそのまま

フリップコード

定数説明
opencv.FLIP_HORIZONTAL左右反転（コード 1）
opencv.FLIP_VERTICAL上下反転（コード 0）
opencv.FLIP_BOTH両軸反転（コード -1）

回転コード

定数説明
opencv.ROTATE_90_CLOCKWISE90° 時計回りに回転
opencv.ROTATE_180180° 回転
opencv.ROTATE_90_COUNTERCLOCKWISE90° 反時計回りに回転

テンプレートマッチングメソッド

定数説明
opencv.TM_SQDIFF二乗差
opencv.TM_SQDIFF_NORMED正規化二乗差
opencv.TM_CCORR相互相関
opencv.TM_CCORR_NORMED正規化相互相関
opencv.TM_CCOEFF相互相関係数
opencv.TM_CCOEFF_NORMED正規化相互相関係数（デフォルト）
