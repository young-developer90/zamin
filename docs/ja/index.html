Zamin

高速で読みやすいプログラミング言語

Zamin は Python ライクな構文と軽量なバイトコード VM を組み合わせています。整数主体のループでは Python に匹敵する速度を実現し、25 以上の組み込み標準ライブラリモジュール、ネイティブ GUI ツールキット、シームレスな C および Python 連携を備え、実用的なプロジェクトに対応します。

Zamin の特徴

⚡

バイトコード VM

レジスタベースのバイトコード VM は、raw ディスパッチ、特化された整数オペコード、ピープホール最適化を備えています。整数ループで CPython に匹敵し、25 以上の組み込みモジュールをすぐに利用できます。

📝

読みやすい構文

最初から自然に感じられる Python ライクな構文。インデントベースのブロック、クリーンな式、わかりやすい型アノテーションにより、コードの読み取りと保守が容易です。

📦

オールインワン

25 以上の標準ライブラリモジュールをすぐに利用可能。ファイル I/O、ネットワーキング、JSON、正規表現、数学、日時など、日常的なタスクに外部パッケージマネージャーは不要です。

🌐

クロスプラットフォーム

単一のコードベースで Linux、macOS、Windows に対応。Zamin コンパイラとランタイムは完全にポータブルで、プロジェクトをどこにでも持ち運べます。

🔌

拡張性

パフォーマンスが重要なコードには C 拡張を記述し、Python ライブラリを直接呼び出し、NVIDIA CUDA で GPU アクセラレーションを利用できます。Zamin は既存のエコシステムと連携します。

🖼

組み込み GUI

Windows では Sol GUI ツールキット、Linux では Luna を使用してネイティブデスクトップアプリケーションを作成。ウィジェット、レイアウト、イベント処理、スタイリングが標準機能として利用できます。

クイック例

Zamin のコードを少しご紹介します：

func greet(name) {
    print("Hello, {name}!");
}

func fibonacci(n) {
    if n <= 1 {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

greet("World");

for i in 0..10 {
    print("fib({i}) = {fibonacci(i)}");
}

パフォーマンス

Zamin のレジスタベースのバイトコード VM は、JIT にヒントを得た最適化により、一般的なワークロードで優れたパフォーマンスを発揮します：

整数ループ（500万回）vs Python 3
0.9x — Python と同等
リスト追加（50万回）vs Python 3
1.5倍遅い
再帰 fib(32) vs Python 3
4.3倍遅い

13.4倍遅い
