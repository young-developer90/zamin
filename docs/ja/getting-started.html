はじめに

このガイドでは、Zamin をソースからインストールし、セットアップを確認し、最初のスクリプトを作成し、大規模なプロジェクトを整理する方法を説明します。

前提条件

Zamin をビルドする前に、システムに以下のものがインストールされていることを確認してください：

  • Rust 1.80+ — Zamin は Rust で書かれています。rustup からインストールしてください：
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Git — リポジトリのクローンに必要です。
  • C コンパイラ — Linux/macOS では GCC または Clang、Windows では MSVC。ネイティブ拡張とオプション機能に必要です。

インストール

クローンとビルド

リポジトリをクローンし、リリースモードで Zamin をビルドします：

git clone https://github.com/young-developer90/zamin.git
cd zamin
cargo build --release

コンパイルされたバイナリは target/release/zamin にあります。便利なように PATH に追加してください：

export PATH="$PWD/target/release:$PATH"

オプション機能

Zamin は追加機能を有効にするオプションの機能フラグをサポートしています：

cargo build --release --features opencv
機能説明
opencv画像処理とコンピュータビジョンのための OpenCV バインディングを有効化
lunaLinux 用の Luna GUI ツールキットを有効化
pythonPython ライブラリを直接呼び出すための Python 連携を有効化

複数の機能を組み合わせることもできます：

cargo build --release --features "opencv,luna,python"

インストールの確認

Zamin がインストールされて動作することを確認します：

$ zamin version
zamin 1.7.3

インタラクティブ REPL を起動して、ランタイムが機能することを確認します：

$ zamin repl
Zamin 1.7.3 (interactive)
Type "help" for more information.
zamin>

最初のスクリプト

hello.zamin というファイルを作成し、以下の内容を記述します：

print("Hello, Zamin!")

実行します：

$ zamin run hello.zamin
Hello, Zamin!

もう少し面白いものを試してみましょう。hello.zamin の内容を次のように置き換えてください：

func greet(name) {
    print("Hello, {name}!");
}

func fibonacci(n) {
    if n <= 1 {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

greet("World");

for i in 0..10 {
    print("fib({i}) = {fibonacci(i)}");
}

REPL の使用

Zamin REPL は、言語機能の実験やコードスニペットのテストに最適です：

$ zamin repl
Zamin 1.7.3 (interactive)
Type "help" for more information.
zamin> print("Hello!")
Hello!
zamin> 2 + 2
4
zamin> func add(a, b) { a + b }
zamin> add(10, 20)
30
zamin> [1, 2, 3].map(func(x) { x * 2 })
[2, 4, 6]
zamin> .exit

プロジェクトセットアップ

大規模なアプリケーションには、Zamin の組み込みプロジェクトスキャフォールディングを使用します：

$ zamin new myproject
Created new project "myproject"
  src/main.zamin
  zamin.json

これにより、以下の構造のプロジェクトディレクトリが生成されます：

myproject/
├── zamin.json
└── src/
    └── main.zamin

zamin.json

zamin.json ファイルはプロジェクトのメタデータとビルド設定を定義します：

{
  "name": "myproject",
  "version": "1.0.0",
  "entry": "src/main.zamin",
  "dependencies": [],
  "features": []
}

src/main.zamin

アプリケーションのエントリポイント：

print("Welcome to myproject!")

プロジェクトをビルドして実行します：

$ cd myproject
$ zamin run
Welcome to myproject!

プロジェクト管理の詳細については、プロジェクト CLI リファレンスを参照してください。
